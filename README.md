# Edusity

Edusity is a modern, responsive educational platform built with ReactJS and Vite,
offering a seamless and dynamic learning experience for students and educators alike.

---

# Features

- Responsive design for mobile, tablet, and desktop
- contact form using wich you can send email.
- Our Programs and campus photos
- Interactive testiminials banner

---

# Tech Stack

- Frontend: ReactJS, Vite
- Contact Management: Web3form API
- Routing: React Router Dom
- Styling: CSS
- Build Tool: Vite
